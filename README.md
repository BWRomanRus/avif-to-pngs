# avif2pngs

Минималистичный конвертер AVIF → PNG / PNG sequence

Небольшая автономная консольная утилита для декодирования AVIF и сохранения результата в PNG (включая извлечение кадров анимации).
Размер бинарника: ~488 KB. Один exe, без установки, без dll, работает офлайн.

## Назначение
Быстро конвертировать .avif:
в один PNG-файл
в папку PNG-кадров (для анимированных AVIF)
извлечь конкретный кадр по индексу
Возможности
AVIF → PNG
Поддержка анимированных AVIF (PNG sequence)
Извлечение кадра --frame N
Один исполняемый файл (~488 KB)
Без внешних зависимостей и подключаемых библиотек (standalone)
Подходит для batch-обработки
Полностью офлайн

## Возможности
- Один исполняемый файл  
- Без библиотек и runtime  
- Извлечение секвенции, конвертирование выбранного кадра, и одиночного изображения  
- Подходит для пакетной обработки

## Недостатки
4K;8K разрешения и выше, обрабатываются дольше обычного, прямая взаимосвязь размера структуры и способа декодирования.
Поддерживаются все стандартные цветовые пространства и типичные профили AVIF.
Редкие и экспериментальные варианты цветовых представлений могут не поддерживаться
